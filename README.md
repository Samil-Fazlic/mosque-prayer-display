# Islamische Gemeinschaft Tevhid e.V. — Gebetszeiten Display

Zweisprachiges (Deutsch/Bosnisch) Gebetszeiten-Display fuer den TV in der Moschee.

## Setup
```bash
npm install
npm run dev
```

## Deployment
Push zu GitHub → Vercel deployed automatisch.

## Anpassungen
In `src/App.jsx` das `MOSQUE`-Objekt bearbeiten.
